## 1. 引入
众所周知，很多程序启动的时候都需要参数，比如 ```g++ -v``` 其中 ```-v```就是参数。```g++ main.cpp -o main.exe``` 中 ```main.cpp```, ```-o```, ```main.exe``` 为附加的三个参数， ```notepad++ %1```中，这个 ```%1``` 经过系统之后也是一个参数<br>
我们的程序同样也可以读取参数，这些启动程序的参数就是 ```main```函数的参数

## 2. main 的参数
```C++
#include <iostream>
using namespace std;
int main(int argn, char** argv){
	cout << argn << endl;
	cout << argv[0];
}
```

```main```函数的参数如上所示。```argn``` 记录参数的数量，```argn```指向参数字符串的数组<br>
值得一提的是，```argv[0]```存储的是程序文件的位置，也就是说，```argn```至少为 1
